# 🚀 Flutter UI Components Collection (GetX Powered)

This repository contains two reusable and scalable frontend Flutter projects built using GetX for state management. The goal is to provide modern, clean, and production-ready UI patterns that can be easily integrated into any Flutter application.

---

## 📦 Projects Included

### 1️⃣ Animated Sidebar Navigation (Extendable Side Menu)

A fully customizable and extendable sidebar navigation system built using:

- sidebar_with_animation
- GetX for state management

![to github1](https://github.com/user-attachments/assets/3c5ae2e3-fb56-4694-b34f-7d70d1e2bd78)

  
#### ✨ Features

- Smooth open/close animations
- Expandable and collapsible menu items
- Dynamic menu configuration
- Responsive design (works on web & desktop)
- Clean separation of UI and logic using GetX
- Easy to plug into any project

#### 🧠 Use Cases

- Admin dashboards  
- CMS systems  
- Enterprise apps  
- Multi-module applications  

---

### 2️⃣ Smart Data Table UI (Advanced Table Design)

A powerful and flexible table implementation using:

- data_table_2
- GetX for reactive state handling
![to github2](https://github.com/user-attachments/assets/4277779e-8081-4fe3-adaf-01a2f59d0d20)

  
#### ✨ Features

- Pagination support
- Sorting & filtering
- Row selection (single & multiple)
- Responsive layout
- Custom cell rendering
- API-ready structure (easy backend integration)
- Optimized performance for large datasets

#### 🧠 Use Cases

- Admin panels  
- Reporting systems  
- Inventory management  
- Financial dashboards
